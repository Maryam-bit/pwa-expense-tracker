## INSTALLATION

1. one the repo
   ```sh
   git clone https://github.com/Maryam-bit/expense-tracker-context-api.git
   ```
2. stall NPM packages
   ```sh
   npm install
   ```
3. Start Project
    ```sh
    npm start
   ```
